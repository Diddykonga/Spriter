# Spriter, Version 0.6
A Plugin for Spriter built for Unreal Engine 4.

Currently Supported Spriter Features:
* Bones
* Sprites
* Character Maps
* Multiple Entities
* Events
* Boxs
* Points

Currently Supported Runtime Features:
* Animation Blending
* Start/End Animation Callback's
* Full Access To Spriter File Data In C++/Blueprints
